## 問題文
$A$ 個のリンゴと $B$ 個のリンゴがあります。  
合わせて何個か求めてください。


## 制約
- $A, B$ は $1$ 以上 $100$ 以下の整数


## 入力
入力は以下の形式で標準入力から与えられます。

$A \ B$


## 出力
答えを標準出力から出力してください。


## サンプル 1
### 入力例 1
```
2 3
```

### 出力例 1
```
5
```

$2$ 個のリンゴと $3$ 個のリンゴがあり、合わせて $5$ 個です。


## サンプル 2
### 入力例 2
```
14 8
```

### 出力例 2
```
22
```
